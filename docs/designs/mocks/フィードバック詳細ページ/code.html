投稿者: yamada
2026/01/15

いつも便利に使わせていただいています。キャラクター一覧画面での検索について要望です。

現在、属性やレベルで絞り込みを行っても、画面を移動するとリセットされてしまいます。キャンペーンの管理中など、同じ条件で何度もリストを確認したい場面が多いため、検索条件をブラウザに保存するか、URLパラメータとして保持してほしいです。

特に「未承認キャラクター」かつ「戦士クラス」のような複合条件を指定するのが毎回手間になっています。ご検討よろしくお願いいたします。

# playwright-test-fixer エージェントの使用方法

## 概要
ユーザーから「テストして」「Playwrightで確認して」などの要求があった場合、このエージェントを使用する。

## 呼び出し方法

```
Task tool:
- subagent_type: playwright-test-fixer
- prompt: 以下の形式で指示を渡す
```

## プロンプト形式

```
URL: http://localhost:3000/[対象ページ]
テスト内容:
1. [操作1]
2. [操作2]
3. [操作3]
期待動作:
- [期待される結果1]
- [期待される結果2]
修正の可否: する/しない（デフォルト: する）
```

## 使用例

### 検索機能のテスト
```
URL: http://localhost:3000/scenarios
テスト内容:
1. 「新クトゥルフ神話TRPG（CoC7版）」ボタンをクリック
2. 「検索」ボタンをクリック
期待動作:
- URLに ?systems=... が追加される
- 検索結果が絞り込まれる（件数が減少）
```

### フォーム送信のテスト
```
URL: http://localhost:3000/scenarios/new
テスト内容:
1. 必須項目を入力
2. 送信ボタンをクリック
期待動作:
- バリデーションエラーなし
- 正常にリダイレクト
```

## 注意事項
- 開発サーバー（localhost:3000）が起動していることを確認する
- テスト対象の機能を明確に指定し、全画面テストは避ける
- 問題発見時は自動修正され、再テストで確認される

# Avatar コンポーネント 要件定義書

**作成日**: 2026-01-26
**TDDフェーズ**: 要件定義
**Pencilデザイン**: docs/designs/scenarios.pen > Components > Avatar/*

---

## 1. 概要

### 1.1 目的
ユーザーのプロフィール画像を表示するアバターコンポーネント。
画像がない場合はフォールバックアイコンを表示する。

### 1.2 使用場所
- ユーザープロフィール
- コメント・レビュー
- 参加者一覧
- 再利用性: 高

### 1.3 Ark UI / 既存コンポーネントとの関係
- ベース: Ark UI Avatar（または独自実装）
- スタイリング: PandaCSS

---

## 2. Props設計

| Prop | 型 | デフォルト | 説明 |
|------|-----|-----------|------|
| `src` | `string` | - | 画像URL |
| `alt` | `string` | - | 代替テキスト |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | サイズ |
| `fallback` | `React.ReactNode` | Userアイコン | フォールバック要素 |

---

## 3. Pencilデザイン詳細

| サイズ | width/height | cornerRadius | アイコンサイズ |
|--------|-------------|--------------|--------------|
| `xs` | 24px | 12px | 12px |
| `sm` | 32px | 16px | 16px |
| `md` | 40px | 20px | 20px |
| `lg` | 48px | 24px | 24px |
| `xl` | 64px | 32px | 32px |

**フォールバック（画像なし）**:
- 背景色: #D1FAE5（淡い緑）
- アイコン色: #10B981（緑）
- アイコン: lucide user

## 4. テスト観点

- [ ] 画像なしでフォールバックが表示される
- [ ] 各サイズが正しく表示される
